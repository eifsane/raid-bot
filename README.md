# Discord Raid Signup Bot

Бот для записи на рейды с ролями, резервом, уведомлениями и веткой обсуждения.

## Что умеет
- `/raid_create` — создать рейд на 5 или 10 человек
- запись ролями: танк / хил / дд / резерв
- авто-резерв, если места закончились
- кнопка отписки
- отдельная ветка обсуждения под каждым рейдом
- уведомление за час до начала
- уведомление в момент начала
- хранение рейдов в `raids.json`

## Railway variables
- `DISCORD_TOKEN` — токен бота
- `GUILD_ID` — ID сервера (необязательно, но удобно для быстрого появления slash-команд)
- `TIMEZONE_OFFSET` — `3` для МСК

## Команды
- `/raid_create`
- `/raid_list`
- `/raid_delete message_id:<id>`
